Heck Food Ltd.

Website Rebuild (2020)

In 2020 I was commissioned to rebuild Heck's website using the Shopify platform. The purpose of using the Shopify platform was so the Heck team could easily manage the site after I had built it. I specifically coded some pages using HTML, CSS and JavaScript, creating special features that Shopify didn't offer.

One of my favourite pages is the 'Veg with Edge' page. This was built by layering a series of images, the background was animated to create a rolling effect and the product information was set to show when the user hovers over the product.
